# Plan: Nâng Cấp Cashflow (Vibe + Economy + Share)
Created: 2026-06-04 12:47
Status: 🟡 In Progress

## Overview
Nâng cấp trò chơi Cashflow với 3 trọng tâm chính:
1. **Game Vibe:** Âm thanh, rung (Haptic), và hiệu ứng pháo giấy (Confetti).
2. **Advanced Economy:** Thị trường nâng cao (Chia tách cổ phiếu) và trả nợ linh hoạt.
3. **Social Share:** Tính năng xuất hình ảnh báo cáo tài chính/kết quả thắng game và chia sẻ lên mạng xã hội.

## Tech Stack
- Frontend: Flutter
- State Management: Riverpod
- Packages bổ sung:
  - `audioplayers` (Âm thanh)
  - `confetti` (Hiệu ứng thắng)
  - `share_plus` (Chia sẻ mạng xã hội)
  - `screenshot` (Chụp ảnh Widget thành Image)
  - `path_provider` (Lưu ảnh tạm)
